# Torrent Scrapper

A responsive web app created using React and BootStrap5 that provides you list of torrents and their magnet link fetched from 1337x.

## Run Locally

Clone the project

```bash
  git clone https://github.com/kishanmodi/Torrent-Scrapper-React
```

Go to the project directory

```bash
  cd Torrent-Scrapper-React/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features

-   Light/dark mode toggle
-   Safe Search Filter
-   Copy Magnet Link/Open in uTorrent
-   Responsive
-   Ad Free
